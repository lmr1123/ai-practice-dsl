# 大参林门店 AI 陪练业务资料整理

## 1. 收银台办卡资料提炼

### 1.1 标准会员权益口径

基于门店原始话术，会员权益可以统一整理为 6 类：

| 权益类型 | 标准表达 |
| --- | --- |
| 免费办理 | 新会员免费办理，不需要充值，不需要实体卡 |
| 当单优惠 | 新办会员可领新人券，本单符合条件时可立减 |
| 新人券 | 常见口径为 3 张 5 元券，合计 15 元；部分门店活动为 3 元、5 元、8 元合计 16 元，或叠加 20 元满减券，需以当期活动为准 |
| 会员价 | 部分商品可享会员价、会员折扣或会员专享活动 |
| 积分权益 | 消费累计积分，积分可兑换礼品、优惠券或按活动规则抵扣 |
| 长期权益 | 会员日、生日券、星级礼、双倍积分、全国门店通用 |

训练时应要求员工先讲“本单能省多少”，再讲“长期有什么用”。不要只说“优惠多、活动多”，必须能说出具体权益。

### 1.2 推荐办卡标准流程

1. 报金额后自然确认会员  
   “您好，请问有大参林会员吗？没有的话现在可以免费办理。”

2. 先讲当单利益  
   “您这单符合新会员券使用条件，办好后这单可以直接减 X 元，同时享会员价/积分。”

3. 再讲长期权益  
   “后续买常备药、日用品也能看会员价，消费有积分，会员日还有活动，大参林全国门店通用。”

4. 说明操作成本  
   “只需要手机号和验证码，大约 30-60 秒，不需要实体卡。”

5. 回应异议  
   根据顾客拒绝原因，只追加一次有针对性的说明。

6. 尊重拒绝并继续收银  
   “好的，理解，那我先帮您正常结账。后续有需要，任意大参林门店都可以办理。”

### 1.3 高频顾客异议

| 异议 | 标准回应 |
| --- | --- |
| 不想留手机号，怕泄露信息 | “理解您的顾虑。手机号主要用于会员账户、积分、优惠券和必要服务通知，我们按公司会员规则保护信息。您后续也可以选择不接收营销类通知。” |
| 怕打骚扰电话/短信 | “理解，很多顾客会担心这个。会员通知主要是权益、券和服务信息，不会要求您额外消费；如果不想接收营销通知，可以按规则关闭或取消。” |
| 赶时间 | “不耽误您，办理大约 30 秒。如果您今天赶时间，我先正常结账，后续随时可以办。” |
| 优惠力度小 | “这单看起来只省几元，但会员价、积分、会员日和新人券是长期权益。您如果常买常备药，累计下来更划算。” |
| 不常来/不住附近 | “大参林会员全国门店通用，路过其他门店也能用。偶尔买一次也可以享当次优惠和积分。” |
| 会员没用/已经有别的会员 | “不强制办理。我们这张主要是药店常备药、会员价、积分和会员日权益，如果您今天想用本单优惠，我可以帮您快速开通。” |
| 不想扫码/嫌麻烦 | “只需要手机号和验证码，不需要实体卡。您不方便的话，我先帮您正常结账。” |

### 1.4 可沉淀为优秀话术的表达

- “不是必须办，您不办也可以正常购买。我只是提醒您这单符合新会员优惠，免费开通后可以直接省 X 元。”
- “办卡不需要充值，主要是手机号绑定会员权益，后续可以查积分、用券和会员价。”
- “您担心被打扰我理解，手机号主要用于会员账户、积分和权益通知，不想接收营销类通知可以关闭。”
- “如果您赶时间，我先帮您结账；后续任意大参林门店都可以免费办理。”
- “优惠不只是这一次，会员价、积分、会员日、星级礼这些是长期权益，常买常备药会更明显。”

### 1.5 风险话术与扣分点

以下原始话术不建议进入标准训练，应作为扣分项：

- “绝对不会联系您”“不会有信息骚扰”：承诺过满，实际不可控。
- “不涉及个人信息”：手机号本身就是个人信息，应改为“按规则保护和使用”。
- “随便填一个手机号”：影响账户真实性和合规。
- “不办会员不能买/没有会员就不给优惠”：容易造成强迫感。
- “用公用会员卡让您享受会员价”：可能带来会员权益、积分归属和合规风险。
- 顾客明确拒绝后反复劝办：影响体验，应最多轻量挽回一次。

### 2.2 商品卖点整理

福尔天山鱼油牛磺酸软胶囊：

- DHA：作为大脑和视网膜相关营养成分，可用于儿童、学生、备考、用脑多等人群的日常营养补充。
- 牛磺酸：可围绕视力、疲劳、用脑压力等做营养支持表达。
- 适用人群：需要改善记忆力的青少年儿童、学生、备考人群、用脑多的人群。
- 联合推荐方向：与钙铁锌、维生素 A、蛋白粉等营养类商品组合，但必须先确认顾客需求。

Y-氨基丁酸多维营养素压片糖果：

- GABA：可围绕睡眠质量、情绪状态、学习压力做营养支持表达。
- L-赖氨酸、钙铁锌、多种维生素：可围绕儿童成长发育、营养补充做表达。
- 适用人群：学习压力大、睡眠质量差、成长发育营养需求明显的人群。
- 核心卖点：高纯度原料、复合营养配方、易吞服、口感好、清洁配方。

### 2.3 合规表达边界

建议说法：

- “这类产品属于营养补充，不是药，主要是补充 DHA、牛磺酸、GABA、赖氨酸、钙铁锌等营养。”
- “如果孩子最近用脑多、学习压力大、睡眠质量一般，可以作为日常营养补充选择。”
- “不同孩子体质和饮食情况不同，效果感受会有差异。”
- “如果存在明显发育迟缓、长期睡眠异常或其他疾病表现，建议先咨询医生或药师。”

禁止说法：

- “吃了肯定长高。”
- “吃了成绩一定提高。”
- “专治近视/失眠/发育慢。”
- “不用看医生，吃这个就行。”
- “所有儿童都适合，不需要问情况。”

### 2.4 新品推荐标准流程

1. 识别机会  
   顾客购买儿童药、学生营养品、护眼产品、钙铁锌、蛋白粉，或提到学习压力、睡眠、视疲劳、成长发育。

2. 需求探询  
   询问孩子年龄、使用目的、饮食睡眠情况、是否正在服用同类营养品、是否有明确疾病或就医建议。

3. 匹配商品  
   用“需求 + 成分 + 场景 + 活动”推荐，不直接硬推。

4. 合规提示  
   明确营养补充属性，不承诺疗效。

5. 成交推进  
   推荐活动组合，如“2 盒更划算”或“加提 5 元/盒”，但不制造焦虑。

## 3. 适合 AI 陪练的客户画像

### 3.1 办卡客户画像

- 赶时间的新客：只关心能不能快速结账。
- 隐私敏感顾客：担心手机号泄露、短信电话骚扰。
- 价格敏感顾客：觉得只省 3-5 元没必要。
- 低频顾客：认为自己不常来，会员没用。
- 老会员疲劳顾客：对各种会员卡反感。

### 3.2 新品推荐客户画像

- 学生家长：孩子学习压力大、用眼多，担心记忆力和视疲劳。
- 备考学生：想买护眼或营养品，但担心智商税。
- 价格敏感家长：觉得 98 元/盒贵，想知道和普通鱼油区别。
- 谨慎家长：担心孩子能不能吃、有没有副作用。
- 功效追问顾客：直接问“能不能长高/提高成绩/改善睡眠”。
